# SignScribe Assets

This directory contains the assets for SignScribe mod.

## Mod Icon

To fix the "broken icon" warning in ModMenu:

1. Create a 128x128 PNG icon file
2. Name it `icon.png`
3. Place it in: `src/main/resources/assets/signscribe/icon.png`

The icon will be displayed in ModMenu and other mod loaders.

## Icon Guidelines

- Size: 128x128 pixels (recommended)
- Format: PNG
- Transparency: Supported (alpha channel)
- Style: Clean, recognizable design representing sign writing/text placement

## Example Icon

You can create a simple icon showing:
- A sign with text on it
- A quill or pen
- A book or scroll
- The letters "SS" (SignScribe)
