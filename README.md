# SignScribe

A Minecraft Fabric mod for easily copying stories from .txth files onto signs.

## Features
- Load pre-formatted .txth files
- Automatically apply text to signs
- Auto-advance to next sign
- Support for all sign types
- Full ModMenu integration
- Commands for control

## Requirements
- Minecraft 1.21.x
- Fabric Loader 0.15.11+
- Fabric API
- Java 21

## Installation
1. Download the .jar file
2. Place in the `mods` folder
3. Launch Minecraft
